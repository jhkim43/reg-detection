# Built-in Map Asset Credits

DeskRPG does not currently bundle third-party built-in office tilesets or
furniture sprite sheets in this directory.

Current state:

- The default office floor and wall tiles are generated in code at runtime.
- Default office objects such as desks, chairs, computers, coffee machines,
  water coolers, reception desks, and cubicle walls are also generated in code
  at runtime.
- Character avatar layers are handled separately under
  [`/assets/spritesheets/`](../spritesheets/) and are documented in:
  - [LPC Character Asset Credits](../spritesheets/CREDITS.md)
  - [LPC Character Asset License Notes](../spritesheets/LICENSE-assets.md)

If third-party built-in map art is added back to DeskRPG in the future, this
file should be updated to list only the asset packs that are actually bundled in
the repository and shipped in application builds.

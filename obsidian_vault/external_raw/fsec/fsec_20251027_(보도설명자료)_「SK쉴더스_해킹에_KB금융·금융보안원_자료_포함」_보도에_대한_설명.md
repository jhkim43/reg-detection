# (보도설명자료) 「SK쉴더스 해킹에 KB금융·금융보안원 자료 포함」 보도에 대한 설명

(보도설명자료) 「SK쉴더스 해킹에 KB금융·금융보안원 자료 포함」 보도에 대한 설명
1기획부2025-10-27

제목 : 「SK쉴더스 해킹에 KB금융·금융보안원 자료 포함」 보도에 대한 설명 (아주경제 등 10.21일자)     1. 기사 내용   □ 아주경제는‘25.10.21일자 「SK쉴더스 해킹에 KB금융·금융보안원 자료 포함」 제하의 기사에서,     ㅇ“금융보안원은 소프트웨어(SW) 구성도, 내부정보제공망, 보안관제망이 업로드됐고”등의 내용을 보도하였습니다.     2. 동 보도에 대한 설명   □ 다크웹에 공개된 금융보안원 관련 자료를 확인한 결과, 유출된 자료는 SK쉴더스가 2021년도 금융보안원의 사업에 참여하기 위해 제안요청서*를 기반으로 자체 생성한 제안서 자료입니다.    * 금융보안원 홈페이지(알림마당 → 입찰공고)에 공개     ㅇ 또한, 당시 SK쉴더스는 해당 사업을 수주하지 못하였으며, 이후 금융보안원이 구축한 구성도와 전혀 무관함을 알려드립니다.   □ 아울러, 금융보안원은 물리적 망분리, 반출입 통제, 암호화 등 강화된 보안통제를 적용하여 내부 자료를 안전하게 관리하고 있습니다. 끝.

Enhancement Technique, 이하 PET)*은 개인정보 원본을 공개하지 않은 상태에서 개인정보를 활용하여 계산·분석 등 정보 처리를 하는 기술로, 데이터 활용과 프라이버시 보호의 균형을 달성        * 합성데이터, 차분프라이버시, 동형암호, 영지식증명, 다자 연산(MPC) 및 연합학습 등이 해당     ㅇ 재식별 위험을 완화하기 위해 개인 식별성 축소, 은닉, 데이터 분할 등의 방식으로 PET를 도입해 침해 위험을 낮추고 안전한 데이터 활용을 지원     ㅇ 주요 PET로 원본 데이터를 모방해 통계적 특성은 같지만 개인을 식별할 수 없는 가상의 데이터를 생성하는 합성데이터, 분산된 데이터 분석 및 AI 모델 학습을 위한 연합학습, 암호화된 상태에서도 연산이 가능한 암호 기술인 동형암호 등이 활용                                        [ PET 주요 분류 ]   구분 종류 특성 개인 식별성 축소 · 파생 데이터 생성형 합성데이터 원본 데이터를 모방해 인공적으로 데이터를 생성 차분프라이버시 개인정보가 포함된 데이터에 개별 정보가 결과에 거의 영향을 주지 않도록 노이즈를 추가 은닉 · 차폐형 동형암호 암호화 상태로 데이터 연산 영지식증명 실제 정보는 공개하지 않되 사실을 증명 데이터 분할 · 접근통제형 다자 연산(MPC) 원본 데이터 공유 없이 공동 계산 연합학습 분산된 데이터 분석 및 AI 모델 학습     □ PET 기술 성숙도가 높아지면서 전 세계적으로 의료, 제조 등 개인정보를 취급하는 모든 데이터 공유 환경에서 활용되고 있으며,     ㅇ 특히 금융은 산업 특성상 다량의 개인(신용)정보를 취급하고 데이터 분석을 통한 부가가치가 높아, 고객 신원확인 및 부정거래 탐지(FDS) 등 목적으로 적극 활용하는 추세                               [ 해외 금융 분야 PET 적용 사례 ]   분야 업무명 (국가/기관) 적용 기술 고객 신원확인(KYC) Sedicii KYC export (아일랜드/Sedicii) 영지식증명 부정거래 탐지 대응(FDS) AI 학습 데이터 생성 (미국/American Express) 합성데이터 Cross-Bank AML 분석 플랫폼 구축(미국/Future Flow Research) 다자 연산(MPC) 금융상품 판매 지원 AI를 활용한 판매 예측 모델 개발 (네덜란드 등/Inpher) 다자 연산(MPC), ㅇ 공유·활용 유형은 데이터를 소유하는 기관의 수에 따라 단독·다자로 구분하고, 데이터 보호조치가 적용되는 시점에 따라 입력 보호(데이터 제공 시)·출력 보호(분석 결과 제공 시)로 구분                          [ 금융 데이터 공유 · 활용 유형별 주요 PET ]   유형 참여 구성 보호 유형 주요 PET 제3자 위탁 분석 단독기관
